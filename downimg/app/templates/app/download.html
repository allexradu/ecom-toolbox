Batch Image Downloader

Do you have a huge list of image URLs you have to download and import into your e-commerce platform?

Al you have to do is specify which columns have the image urls, and hit download and you'll get a zip file that has both the Excel file with all the photos files name in their respective columns and a folder with all your images, ready to import into your e-commerce platform.

Instructions:

Let's say you have an Excel Spreadsheet with four columns of image urls and you want to get another four columns of just image filenames. Download Example.

We use numerical value for columns. (e.g. A-column = 1, B-column = 2, C-column = 3, D-column = 4, E-column = 5 ...)

Steps:

  1. You type the first columns that contains urls, in our example is 1 (A-column)
  2. You type the last column that contains urls, in our example is 4 (D-column). If you only have one column, the last column will be the same as the first.
  3. You type the first columns that you want to be completed with image file names, in our example is 5 (E-column)
  4. You type the last column you want to be completed with the image file names, in our example is 8 (H-column). If you only have one column, the last column will be the same as the first.
  5. You choose the XLSX file name (has to be less than 10MB)
