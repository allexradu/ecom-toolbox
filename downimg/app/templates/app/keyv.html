Instructions:

Let's say you have a with 9 column file, the product code as A-column, and four pairs of key-value pairs,see the example to get understand quickly what we are writing about. Download Example.

Steps:

  1. Place you key value as alternate columns (key1 = B-Column, value1 = C-Column, key2 = E-Column, value2 = F-column ...
  2. Make sure first column in range is a key.
  3. Type first key column (in our example is the column B, so type 2, because is se second column, we only accept numerical values).
  4. Type the last value column (in our example is column I, so type 9).
  5. Type the first empty cell were you want the software begin writing your data (in our example is column J, so type 10).
  6. Upload your excel file.
  7. Press start.
  8. Download your processed file.
